# 🎯 Python Number Guessing Game

An interactive, color-coded console game built with Python. Challenge your intuition by guessing the secret number generated by the computer across various difficulty levels!



## ✨ Features

- **Multiple Difficulty Levels:**
  - `Easy`: Numbers between 0-10.
  - `Medium`: Numbers between 0-50.
  - `Hard`: Numbers between 0-100.
- **Custom Game Mode:** Take full control! Set your own lower and upper limits and choose how many lives (attempts) you want to have.
- **Colored Feedback:** Uses ANSI escape codes to provide intuitive feedback (e.g., yellow for "higher", red for "smaller") to enhance the player experience.
- **Replayability:** Built-in loops allow you to jump back to the menu or restart a level immediately after a win or loss.

## 🚀 How to Run

1. Make sure you have [Python](https://www.python.org/) installed on your system.
2. Download the `main.py` file.
3. Open your terminal or command prompt in the folder where the file is located.
4. Run the game using the following command:
   ```bash
   python main.py
